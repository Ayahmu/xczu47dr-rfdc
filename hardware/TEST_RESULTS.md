## Issues and Resolutions

### Issue 1: Mill Command Not Found
**Problem**: Original script used `mill chisel.runMain led.LedTop`
**Error**: `Cannot resolve chisel.runMain`
**Root Cause**: No module named "chisel" in build.sc
**Solution**: Changed to `mill led.run` and `mill gpio.run`
**Status**: ✅ RESOLVED

### Issue 2: Missing Main Class
**Problem**: `ClassNotFoundException: led.elaborate`
**Root Cause**: No elaborate object in led/gpio modules
**Solution**: Created elaborate.scala for each module with ChiselStage
**Status**: ✅ RESOLVED

### Issue 3: Generated Files in Git
**Problem**: .fir and .anno.json files were staged for commit
**Root Cause**: .gitignore didn't exclude these file types
**Solution**: Updated .gitignore to exclude *.fir and *.anno.json
**Status**: ✅ RESOLVED

### Issue 4: Nested ILA Sub-design Error
**Problem**: Block Design generation fails for system_ila_0_0
**Error**: `Nested sub-design can only be generated by its parent`
**Root Cause**: ILA (Integrated Logic Analyzer) debug IP has nested BD
**Impact**: Non-critical, project still usable
**Status**: ⚠️ KNOWN ISSUE (does not affect basic functionality)
